setData(dataSource)

Instantiates the data source pipeline.

Each new layer is created using the supplied constructor and a reference to the previous data source in the pipeline. A reference to each new layer is added to this dataModel as a property using the layer's name.

The first layer must have a @@CLASS_NAME of 'DataSource'. Hence, the start of the pipeline is this.source. The last layer is assigned the synonym this.dataSource.

Branches are created when a layer specifies a name in parent.

Parameters:
Name Type Description
dataSource Array.<object>

Array of uniform objects containing the grid data.

setFilter(columnIndexOrName, stateopt, optionsopt) → {undefined|Error|string}

Set a particular column filter's state.

After setting the new filter state, reapplies the filter to the data source.

Parameters:
Name Type Attributes Default Description
columnIndexOrName number | string

The column filter to set.
state string | object <optional>

A filter tree object or a JSON, SQL, or CQL subexpression string that describes the a new state for the named column filter. The existing column filter subexpression is replaced with a new node based on this state. If it does not exist, the new subexpression is added to the column filters subtree (filter.columnFilters).

If undefined, removes the entire column filter subexpression from the column filters subtree.
options FilterTreeSetStateOptionsObject <optional>

Passed to the filter's setState method. You may mix in members of the FilterTreeValidationOptionsObject
options.syntax string <optional>
 'CQL'

The syntax to use to describe the filter state. Note that setFilter's default syntax, 'CQL', differs from the other get state methods.
Returns:

undefined indicates success.

setGlobalFilterCaseSensitivity(isSensitive)

Set the case sensitivity of filter tests against data.

Case sensitivity pertains to string compares only. This includes untyped columns, columns typed as strings, typed columns containing data that cannot be coerced to type or when the filter expression operand cannot be coerced.

NOTE: This is a shared property and affects all grid managed by this instance of the app.
